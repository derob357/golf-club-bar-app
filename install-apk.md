# Install DHGC App on Samsung Galaxy S25 Ultra

## APK Location
```
/Users/drob/Documents/DHGC/android/app/build/outputs/apk/release/app-arm64-v8a-release.apk
```

## Installation Methods

### Method 1: USB Transfer (Recommended)
1. Connect your Samsung S25 Ultra to your Mac via USB
2. Open Android File Transfer (or use Finder if your Mac supports it)
3. Copy the APK file to your phone's Downloads folder
4. On your phone:
   - Open the Files app
   - Navigate to Downloads
   - Tap `app-arm64-v8a-release.apk`
   - Allow installation from unknown sources if prompted
   - Tap Install

### Method 2: Email
1. Email the APK to yourself
2. Open the email on your Samsung S25 Ultra
3. Download the attachment
4. Tap the downloaded APK to install

### Method 3: Cloud Storage (Google Drive, Dropbox, etc.)
1. Upload APK to your cloud storage
2. Download from cloud storage on your phone
3. Tap to install

## Enable Unknown Sources
If you get a security warning:
1. Settings → Security → Install unknown apps
2. Select the app you're installing from (Files, Gmail, Chrome, etc.)
3. Toggle "Allow from this source"

## Troubleshooting
- **"App not installed"**: Make sure you selected the arm64-v8a version
- **"Parse error"**: The APK may be corrupted, rebuild it
- **Can't find APK on phone**: Check Downloads folder or use a file manager app
